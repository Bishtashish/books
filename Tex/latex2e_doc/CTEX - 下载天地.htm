MiKTeX 1.20e for Windows (22427K)

　　Windows 下最好用的 TeX 系统之一。安装使用都很方便。它所带的 DVI 预览器 Yap 也是最棒的。WinEdt 的缺省设置即是针对 MiKTeX 的。升级较快，一些较新的 TeX/LaTeX 工具如 pdftex/pdflatex, dvipdfm 等都有最新版本。大小适中，推荐使用。

fpTeX 0.4 for Windows

　　teTeX 的 Windows 版本，功能强大。提供大量工具包。并集成了 GhostScript、GSView、WinEdt、ImageMagic 等实用程序。升级也很快。唯一缺点就是比较大。如果下载方便的话，还是推荐使用。
